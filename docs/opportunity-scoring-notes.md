# Opportunity Scoring Notes

## Goal
Use form responses to estimate lead intent and prioritize follow-up.

## Suggested Scoring
- Ready Now = +40
- Ready Soon = +20
- 1 to 3 years dealing with issue = +20
- 3+ years dealing with issue = +25
- Currently using medication = +15
- Open to herbal protocol = +20

## Lead Temperature
- Hot = 60+
- Warm = 35–59
- Cold = 0–34

## Use
This can be tracked manually in GHL or added through workflow logic and custom fields.
