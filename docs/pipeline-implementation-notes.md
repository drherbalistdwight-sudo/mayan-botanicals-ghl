# Pipeline Implementation Notes

## Pipeline Name
Wellness Client Pipeline

## Stages
1. New Lead
2. Assessment Completed
3. Hot Lead
4. Consultation Booked
5. Consultation Completed
6. Protocol Sent
7. Closed Won
8. Closed Lost
9. Nurture

## Rules
- Move new form submissions into Assessment Completed
- Move high-intent leads into Hot Lead
- Move booked contacts into Consultation Booked
- Move non-responsive leads into Nurture if needed

## Purpose
Track each lead from assessment to consultation and final outcome.
