## Recommended Logo Variants
- Main logo with icon + wordmark
- Light version for dark backgrounds
- Dark version for light backgrounds

## Usage Notes
- Use the main logo in headers, footers, and primary brand sections
- Keep clear spacing around the logo
- Avoid stretching or distorting the logo
- Prefer SVG where possible for crisp display
