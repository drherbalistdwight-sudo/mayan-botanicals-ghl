# Brand Colors

## Primary Colors
- Background Dark: #1E2021
- Background Alt: #252829
- Panel Dark: #2D3032
- Yellow Accent: #FFFB00
- Green Primary: #4F7A28
- Green Light: #6AAA35
- Lime Accent: #7EC320
- Tan Text: #D3CFCA
- White: #FFFFFF

## Usage Notes
- Use yellow for key CTA emphasis and headings
- Use green for trust, wellness, and support accents
- Use tan for readable body copy on dark backgrounds
- Keep backgrounds dark for premium visual consistency
