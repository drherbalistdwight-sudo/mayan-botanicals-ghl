# Mayan Botanicals GHL Build

This repository contains the source files, copy, workflow maps, prompts, and deployment assets for the Mayan Botanicals GoHighLevel funnel system.

## Core Funnel
Landing Page → Assessment → Booking → Thank You

## Main Goals
- Generate qualified wellness leads
- Book free 15-minute consultations
- Maintain FTC-compliant messaging
- Build a scalable wellness funnel system

## Key Directories
- /copy = approved messaging
- /pages = GHL-ready HTML
- /forms = field specs
- /workflows = automations
- /prompts = Claude prompts
- /exports = final packaged builds

## Deployment Target
GoHighLevel Funnels, Forms, Workflows, CRM
